Desktop Voice Assistant

A voice-controlled desktop assistant built with Python, enabling real-time interaction using speech recognition and text-to-speech. Supports everyday commands like file handling, system tasks, and quick information retrieval.

Overview

What it does

The assistant was designed to respond to voice commands and make desktop interactions feel more natural. It combines speech recognition, text-to-speech, and command execution in a way that feels practical and immediate.

Key features

Core functionality

  • Real-time voice command handling using Python-based speech recognition.
  • Text-to-speech responses for conversational feedback.
  • Features like weather updates, quick information lookup, and file handling.
  • OpenAI-powered search support for relevant and fast information retrieval.

Why it matters

Project impact

This project helped demonstrate how AI-assisted interaction can make desktop workflows feel more intuitive. It also strengthened my understanding of Python scripting, user interaction design, and integrating smart features into practical tools.
